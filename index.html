What is Simon:

Simon is a memory game that was made in 1978. It is a basic and very easy to play game, it consists of a round base which has four coloured buttons on the top for the user to press. When playing the game the user has to remember the order that the buttons flash. It is a game that can be enjoyed at all ages. Even though the game is simple it is also challenging and can become a competition amongst users, Whether it be friends or stangers users can see if they can beat other users.
