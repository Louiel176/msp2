How to play:

How to play Simon: Simon is a memory game that originally consisted of four coloured buttons that would flash in a sequence for the user to memorise and repeat back to the game. The game would start with one button flash for the user to repeat back to the game and if the user was correct the game would continue by adding another button to the sequence this can be either a new button or the same one that flashed previously. The game you are about to play will consist of 5 levels. The levels will add more buttons onto the game allowing the user to have a more challenging experience. Level 1 will consist of 4 buttons like the original game and each level on from that will add an extra button so level 5 will consist of 8 buttons. There is no limit to reach therefore the game may go on inifinitely to end the game you must lose.
